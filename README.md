📌 Overview
This project presents an interactive Tableau dashboard analyzing football (soccer) league performance. It focuses on team rankings, match outcomes, and key performance indicators such as wins, losses, and draws.


------
🎯 Objectives
Analyze league standings and team performance
Compare clubs based on match outcomes
Highlight top-performing teams dynamically
Provide insights into home vs away performance
------
📂 Dataset
Source: European Soccer Dataset (Kaggle)
Includes match-level data, team stats, and league information
------

🛠️ Tools Used
Tableau Desktop
Tableau Prep (for data cleaning & transformation)
GitHub (project publishing)

------

📈 Key Features
🏆 League standings table with dynamic ranking
🎯 KPIs: Wins, Losses, Draws
🔥 Highlighting Top 2 teams
⚽ Home vs Away performance analysis
🎨 Clean and interactive dashboard design
